Features

Structure Analysis

Parse and understand document structure including headings, paragraphs, lists, and semantic sections for targeted transformations.

Paragraph Restructuring

Intelligently reorganize paragraph structures while maintaining logical flow and argument coherence across the document.

Sentence Reflow

Rebuild sentence structures to improve readability, vary cadence, and eliminate repetitive patterns without changing meaning.

Multi-Language

Full support for Chinese and English text processing with language-aware restructuring algorithms and tokenization.

Basic Text Restructuring

from openorm import OpenORMClient

client = OpenORMClient(api_key="your-api-key")

# Restructure a paragraph
result = client.restructure(
    text="""The rapid advancement of AI technology has led to
    significant changes in content creation. Many organizations
    now use AI to generate initial drafts.""",
    mode="natural",
    preserve_keywords=True
)

print(result.restructured_text)
print(f"Similarity score: {result.similarity:.0%}")

Document-Level Processing

# Process an entire document with structure preservation
doc_result = client.process_document(
    text=long_document,
    options={
        "preserve_headings": True,
        "restructure_depth": "paragraph",  # or "sentence"
        "language": "auto"
    }
)

for section in doc_result.sections:
    print(f"[{section.type}] {section.text[:80]}...")
